# React Todo App

A simple and modern Todo application built using **React**, **Context API**, and **LocalStorage**. This app helps users manage their daily tasks efficiently with features like adding, editing, deleting, and marking tasks as completed.

## 🚀 Features
- Add new todos
- Edit existing todos
- Delete todos
- Mark todos as completed
- Persistent data using LocalStorage
- Global state management using Context API
- Responsive UI using Tailwind CSS

## 🛠️ Technologies Used
- React
- Context API
- Tailwind CSS
- JavaScript (ES6)
- LocalStorage
- Vite
